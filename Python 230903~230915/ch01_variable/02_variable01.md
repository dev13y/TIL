# 02_variable01

## `Python<02_variable01>`
```py
result = 10
print(result)

result = 20
print(result)

#  여러 개의 변수에 여러 개의 값 저장
a, b, c = 1, 2, 3
print(a)
print(b)
print(c)

# 여러 개으 변수에 한 개의 값 저장ㅇ
a = b = c = 10
print(a)
print(b)
print(c)

# 두 변수의 값 교환
a, b = 10, 20
print(a, b)
a, b = b, a # 교환
print(a, b)

# 변수 삭제 : del
a = 10
del a
print(a)
```
