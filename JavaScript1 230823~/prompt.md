# prompt

## `JavaScript<prompt>`
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>prompt() 함수</title>
    <script>
        // 변수 선언하고 prompt() 함수에서 입력한 값을 반환하면 변수에 저장
        let answer = prompt("가장 좋아하는 과일은?");
        document.write("가장 좋아하는 과일은 " + answer + " 입니다");
    </script>
</head>
<body>
    
</body>
</html>
```
