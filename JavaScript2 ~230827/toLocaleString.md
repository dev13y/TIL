# toLocaleString

## `JavaScript<toLocaleString>`
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>toLocaleString()</title>
    <script>
        // 천단위 구분 표시 : toLocaleString()
        let amount = 1000000;
        document.write(amount + "<br>");
        document.write(amount.toLocaleString() + "<br>");

        // 소수점 자릿수 설정 : toFixed(자릿수)
        let avg = 83.33333333;
        document.write(avg.toFixed(2));
    </script>
</head>
<body>
    
</body>
</html>
```
