# function_hoisting

## `JavaScript<function_hoisting>`
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>함수 호이스팅</title>
    <script>
        // 함수 호이스팅 : 함수가 선언(정의) 되기 전에
        // 함수를 호출하는 것
        show(); // 함수 호이스팅 가능

        function show() {
            alert("shwor() 함수입니다");
        }
    </script>
</head>
<body>
    
</body>
</html>
```
