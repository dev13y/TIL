## `CSS<list>`
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>목록 관련 속성</title>
    <!-- <ul> 디폴트 : type="disc" -->
    <style>
        /* ul {list-style-type: square;} */
        /* ul {list-style-type: none;} */
        ul li { display: inline;} /* 블릿기호 없이 옆으로 배치 */
    </style>
</head>
<body>
    <ul>
        <li>봄1</li>
        <li>여름2</li>
        <li>가을3</li>
        <li>겨울4</li>
    </ul>
</body>
</html>
```
